<objective>
In the GizzVerse constellation (`ConstellationCanvas.tsx`), song-name labels are
drawn inside `nodeCanvasObject` (block "5. Label", ~lines 448-468). Because
react-force-graph-2d paints nodes one-at-a-time (fill + label per node), a label
drawn for an earlier node can be overpainted by a LATER node's orb fill, and
labels interleave with orbs and never guarantee to sit above the edge lines.

Fix: move ONLY the text-label drawing into a dedicated top pass via
`onRenderFramePost(ctx, globalScale)` — the hook that runs AFTER all links, all
arrows, and all nodes render, in the same world-coordinate transform. This
guarantees every visible label paints on top of everything else.

Purpose: Labels are always legible over dense edge clusters and neighboring orbs.
Output: Refactored `ConstellationCanvas.tsx` — a `drawNodeLabel` helper + wired
`onRenderFramePost`, label block removed from `nodeCanvasObject`. Pure render
change; zero behavior change beyond z-order.
</objective>

`nodeVisibility` filter that `nodeCanvasObject` benefits from:
  1. Return early if `node.x == null || node.y == null`.
  2. Return early if `!isNodeVisible(node.id)` (replicates the nodeVisibility gate
     the node pass got for free — do NOT relabel filtered-out nodes).
  3. Recompute `focused = node.id === focusId` and
     `neighbor = focusId != null && neighborIds.has(node.id)`.
  4. Recompute the depth-scaled radius `r` for the label y-offset — same formula
     as the node pass: `radiusFor(node.playCount) * lerp(dz.DEPTH_RADIUS_FAR,
     dz.DEPTH_RADIUS_NEAR, z)` where `z = node.z` and `dz = config.explore.depth`.
  5. Recompute the combined alpha EXACTLY as the node pass does: `focusAlpha`
     (1 if `isInNeighborhood(node.id)` else `config.explore.FOCUS_DIM_OPACITY`);
     `dexAlpha` — replicate the dex-dim path: `sightings = overlay && sightingsFor
     ? sightingsFor(node.id) : 0`, `unseen = overlay && sightings === 0`, then
     `dexAlpha = unseen ? config.explore.DEX_DIM_OPACITY : 1`; `depthAlpha =
     lerp(dz.DEPTH_OPACITY_FAR, dz.DEPTH_OPACITY_NEAR, z)`; final =
     `Math.max(dz.DEPTH_ALPHA_FLOOR, Math.min(focusAlpha, dexAlpha) * depthAlpha)`.
     Wrap in `ctx.save()` / set `ctx.globalAlpha` / ... / `ctx.restore()`.
  6. Recompute the `showLabel` gate identically: `forced = focused || neighbor`;
     `showLabel = forced || globalScale >= config.explore.LABEL_ZOOM_THRESHOLD ||
     topKIds.has(node.id)`; return early if `!showLabel`.
  7. Keep font/color/ellipsize/fillText byte-identical to the current block:
     `fontPx = (forced ? 14 : 12) / globalScale`; `ctx.font = `${forced ? "600 "

     focused ? node.name : ellipsize(node.name, config.explore.LABEL_MAX_CHARS)`;
     `ctx.fillText(text, node.x, node.y + r + 2 / globalScale)`.

Wire the pass on `<ForceGraph2D>` by adding the prop
