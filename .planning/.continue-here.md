## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Relative paths after a persisted `cd` | The Bash tool's working directory persists across calls: after `cd packages/app && npm run build`, later relative-path checks (`ls .planning/...`) silently ran from `packages/app` and produced a false "todo file missing on disk" conclusion that got committed into STATE.md before being caught. | advisory | Use absolute paths (or re-`cd` to repo root inside the same command) for any file-existence claim; treat "file not found" via relative path as unverified until confirmed from a known cwd. |
| Tunneling vite preview directly | `vite preview` rejects unknown hostnames (`preview.allowedHosts`) — a cloudflared tunnel to 4175 returned 403s. | advisory | Tunnel a plain static server over `dist` instead (scratchpad `serve-dist.cjs` pattern), or add the host to preview.allowedHosts if a repo change is warranted. |
