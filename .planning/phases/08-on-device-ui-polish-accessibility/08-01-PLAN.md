## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01 | Tampering/Elevation | `<Sheet>` children rendering | mitigate | Primitive passes `children` through untouched; injects NO HTML, never `dangerouslySetInnerHTML`. Preserves the app-wide React-text/`fillText`-only invariant. |

| T-08-SC | Tampering (supply chain) | npm installs | mitigate | This plan (and phase) installs ZERO new runtime packages; the a11y layer is dependency-free hooks on React 19 native `inert` + baseline DOM. No install task, no legitimacy gate needed. |
